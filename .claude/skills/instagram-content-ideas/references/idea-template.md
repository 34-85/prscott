# Content-idea package template

Produce this for every save. Keep it tight and production-ready — a solo founder
should be able to hand any platform section to a shoot/edit without more thinking.

## Presentation format (in chat, for review)

```
### Idea N — from @{author} ({content_type})
Source: {url}
Reframed angle: {one sentence — the transferable principle for AI solo founders}

**Hooks**
1. {contrarian / myth-busting angle}
2. {curiosity-gap / question angle}
3. {outcome-driven / result angle}

**Outline**
- Hook: {the chosen opening beat}
- Key Points:
  - {point 1}
  - {point 2}
  - {point 3}
- CTA: {one clear ask — follow, comment a keyword, grab a template, etc.}

**Platforms**
- Instagram: {Reel or carousel? length, on-screen text, first-frame, caption idea}
- TikTok: {hook in first 2s, native/casual framing, sound/trend note, length}
- YouTube: {Short vs long-form, title, thumbnail concept, chapter beats if long}
```

## Notion page format (when saving an approved idea)

Create the page in the **Content Ideas** database.

**Properties**
- `Name` (title): a short idea title, not the raw caption.
- `Source URL` (url): the Instagram permalink.
- `Source Author` (rich text): the original creator's handle.
- `Content Type` (select): `Post` / `Reel` / `Carousel` (mirror the save).
- `Platforms` (multi-select): `Instagram`, `TikTok`, `YouTube`.
- `Status` (select): `Idea`.

**Page body** (blocks) — use headings and bullets:

```
## Reframed angle
{one paragraph}

## Hooks
1. …
2. …
3. …

## Outline
**Hook:** …
**Key Points:**
- …
- …
- …
**CTA:** …

## Platform breakdowns
### Instagram
…
### TikTok
…
### YouTube
…
```

## Quality bar

- Hooks are one line each and genuinely distinct — not three rewordings.
- Key points are concrete (name a tool, a number, a step), not "leverage AI".
- CTA is a single action.
- Platform sections respect native norms: TikTok = fast/casual/sound-led,
  Instagram = polished/visual/saveable, YouTube = search-title + retention beats.
